# AI x Solana Resources

A curated list of AI tools, frameworks, SDKs, and projects at the intersection of AI and Solana.

## AI Agent Frameworks

| Name | Description | Link |
|------|-------------|------|
| **Solana Agent Kit** | Official Solana toolkit for building AI agents that can interact with Solana protocols | [GitHub](https://github.com/sendaifun/solana-agent-kit) |
| **ElizaOS** | Multi-agent simulation framework with Solana plugin support | [GitHub](https://github.com/elizaOS/eliza) |
| **GOAT (Great Onchain Agent Toolkit)** | Framework for building onchain AI agents across multiple chains including Solana | [GitHub](https://github.com/goat-sdk/goat) |
| **Rig** | Rust framework for building LLM-powered applications with Solana integrations | [GitHub](https://github.com/0xPlaygrounds/rig) |
| **ZerePy** | Python framework for deploying AI agents on Solana and other chains | [GitHub](https://github.com/blormdev/ZerePy) |
| **solagent.rs** | Rust-based toolkit for building Solana AI agents | [GitHub](https://github.com/solana-agent/solagent.rs) |
| **Breeze** | Lightweight AI agent framework designed for Solana DeFi interactions | [GitHub](https://github.com/pyanoxyz/breeze) |
